    \hypertarget{problem-1-separability}{%
\subsubsection{Problem 1: Separability}\label{problem-1-separability}}

    Definition \$\textbackslash{}textit\{convex\_hull \}: \$

\(x=\sum_{n} \alpha_nx_n\) , where \$\alpha\_n \geq 0 \$ and
\(\sum_n \alpha_n = 1\)

Consider a second set of points \{\(x'_m\)\} with a corresponding convex
hull. Then the two sets of points will be linearly separable if there
exists a vector \(\textbf{w}\) and a scalar \(w_0\) such that:

\(w^{T}x_n + w_0 > 0\), for all \(x_n\) and \$ w\^{}\{T\}x\_n'+ w\_0
\textless{} 0\$

\textbf{Prove that if two sets of points are linearly seperable, their
convex hulls do not intersect:}

** Simplify Givens: **

For the set of points in \({\{x^n\}}\) it will correspond to:

\begin{itemize}
\tightlist
\item
  \(w^{T}(\sum_{n} \alpha_nx_n) + w_0 > 0\)
\end{itemize}

Using the definition of the convex hull (Since \(\sum_n \alpha_n = 1\))
, we can simplify this to:

\begin{itemize}
\tightlist
\item
  \(\sum_{n}\alpha_n(w^{T}x_n + w_0) > 0\)
\end{itemize}

Similarly, for the set of ponts in \(\{x_n'\}\) it will correspond to: *
\(\sum_{n}\alpha_n(w^{T}x_n'+ w_0) < 0\)

** Proof:** * Let's assume that the convex hulls intersect.

This means there exists a point, \(x_{nm}\) \(\in\) \(\{x_n\}\) \&
\(\{x_n'\}\). Which implies that at \(x_{nm}\) :

\$\sum\emph{\{n\} \alpha\emph{n(w\^{}\{T\}x}\{nm\} + w\_0) = \sum}\{n\}
\alpha\emph{n(w\^{}\{T\}x}\{nm\} + w\_0) \$

However this is impossible to simultaneously satisfy if they are
linearly separable, since: \(\sum_{n} \alpha_n(w^{T}x_{nm} + w_0)> 0\)
\& \$\sum\_\{n\} \alpha\emph{n(w\^{}\{T\}x}\{nm\} + w\_0) \textless{} 0
\$

Thus this implies that if the convex hulls intersect then their sets are
not linearly separable. Which by the \textbf{contrapositive proves that
if two sets of points are linearly separble then their convex hulls do
not intersect}.

    \hypertarget{problem-2-logistic-regression-and-gradient-descent}{%
\subsubsection{Problem 2: Logistic regression and gradient
descent}\label{problem-2-logistic-regression-and-gradient-descent}}

    \hypertarget{problem-2a}{%
\paragraph{Problem 2a:}\label{problem-2a}}

\(\sigma{'(a)}\) = \(\frac{\partial}{\partial{a}}(1+e^{-a})^{-1}\)

\(= -(1+e^{-a})^{-2} (-e^{-a})\)

\(= -(1+2e^{-a} + e^{-2a})^{-1} (-e^{-a})\)

\$=\frac{e^{-a}}{(1+2e^{-a} + e^{-2a})} \$

Thus: \$\sigma{'(a)} = \frac{e^{-a}}{(1+2e^{-a} + e^{-2a})} \$

\textbf{Now does this mean:}

\(\sigma{(a)}(1-\sigma{(a)}) \stackrel{?}{=} \sigma{'(a)}\)

\(=(\frac{1}{(1+e^{-a})})(1-\frac{1}{(1+e^{-a})})\)

\$=\frac{(1+e^{-a})-1}{(1+2e^{-a} + e^{-2a})} \$

\$=\frac{e^{-a}}{(1+2e^{-a} + e^{-2a})} \$

This proves that \textbf{yes},

\(\sigma{(a)}(1-\sigma{(a)}) \stackrel{\checkmark}{=} \sigma{'(a)}\)

    \hypertarget{problem-2b}{%
\paragraph{Problem 2b:}\label{problem-2b}}

    Compute first derivative:

\(\frac{\partial{L\{x^{(i)},y^{(i)}\}\stackrel{n}{i=1} }}{\partial{w_j}}\)

\$=\frac{\partial}{\partial{w}}\sum\_\{i=1\}\^{}\{n\}
\frac{-y_i}{\sigma{(w^Tx)}} x\sigma{(w^Tx)}(1-\sigma{(w^Tx)}) -
\frac{(1-y_i)}{1-\sigma{(w^Tx)}} (-x)\sigma{(w^Tx)}(1-\sigma{(w^Tx)}) \$

\(=\sum_{i=1}^{n} -x_iy_i(1-\sigma{(w^Tx)}) + x_i(1-y_i)\sigma{(w^Tx_i)}\)

\(=\sum_{i=1}^{n} -x_iy_i + x_i\sigma{(w^Tx_i)}\)

Thus: \$\frac{\partial}{\partial{w}} = \sum\_\{i=1\}\^{}\{n\} -x\_iy\_i
+ x\_i\sigma{(w^Tx_i)} \$

    \hypertarget{problem-2c}{%
\paragraph{Problem 2c:}\label{problem-2c}}

To prove convexity, the second derivative
\(\frac{\partial^2{L}}{\partial{w^2}} \geq 0\) (The Hessian Matrix is
PSD)

Thus computing the second derivative:

\(\frac{\partial^2{L}}{\partial{w^2}}\) \$=\frac{\partial}{\partial{w}}
